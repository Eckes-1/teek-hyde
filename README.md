# vitepress-theme-teek

在线主题文档：[Teek](http://vp.teek.top/)。

## 本地运行

项目拉取

```bash
git clone https://gitee.com/SeasirEckes/teek-hyde.git
```

依赖安装（只能用 pnpm 安装依赖）

```bash
pnpm install
```

## 项目启动

```bash
pnpm docs:dev
```

## 项目打包

```bash
pnpm docs:build
```

## 升级

```bash
pnpm add vitepress-theme-teek@latest -D
```
