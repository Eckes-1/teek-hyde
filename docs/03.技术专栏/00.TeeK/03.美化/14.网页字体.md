## 下载

个人喜欢筑紫A丸的字体，所以下载的是的[筑紫A丸](https://github.com/Zolyn/TsukuARdGothic-Std/blame/main/renamed/TsukuARdGothicStd-Bold.ttf)，其他的字体可以自行下载。

::: tip 字体下载

- [筑紫A丸](https://github.com/Zolyn/TsukuARdGothic-Std/blame/main/renamed/TsukuARdGothicStd-Bold.ttf) <Badge type="tip" text="推荐" />

- [苹方字体](https://github.com/Lubohuan/fontstyle-pingfang/tree/master)

- [阿里巴巴普惠体](https://www.alibabafonts.com/)

:::

## 配置

- 创建`Demo\docs-base\public`目录，将下载的字体文件放入`fonts`目录。

```md{5-7}
.
├─ Demo
│ └─ docs-base
│ │ └─ public
│ │ │ └─ fonts
│ │ │ │ └─ 筑紫A丸
│ │ │ │ │ └─ TsukuARdGothicStd-Bold.ttf  >>> 字体文件
└─ node_modules

```

- 在`Demo\docs-base\.vitepress\theme\style\`目录下，创建`font.scss`文件，添加以下内容：

```scss{3-7,15-16}
/* 定义自定义中文字体 */
@font-face {
  font-family: "筑紫A丸";
  src:
    /* 优先尝试从远程 CDN 加载（按格式优先级排序） */
    url("https://cdn.jsdelivr.net/gh/Seasir-Hyde/live2d-widget@v1.0.2/fonts/Roboto-Regular.ttf") format("truetype"),
    url("https://cdn.jsdelivr.net/gh/Seasir-Hyde/live2d-widget@v1.0.2/fonts/Roboto-Regular.ttf") format("woff2"),
    url("https://cdn.jsdelivr.net/gh/Seasir-Hyde/live2d-widget@v1.0.2/fonts/Roboto-Regular.ttf") format("woff"),
    /* 远程加载失败后，回退到本地字体 */
    url("/fonts/TsukuARdGothicStd-Bold.ttf") format("truetype"),
    url("/fonts/TsukuARdGothicStd-Bold.ttf") format("woff2"),
    url("/fonts/TsukuARdGothicStd-Bold.ttf") format("woff");
  font-weight: 300; //定义字体的粗细程度，300 表示相对较细的字体
  font-style: normal; //定义字体的样式，normal 表示正常样式，无倾斜效果
  font-display: swap; //字体显示策略，swap 表示在字体加载完成前使用系统默认字体，加载完成后替换为自定义字体
}

/* 应用字体 */
:root {
  --vp-font-family-base: "筑紫A丸", "筑紫A丸", sans-serif; /* base：影响正文、标题等文本。 */
  --vp-font-family-mono: "JetBrains Mono", "Cascadia Mono", "Not Sans SC", monospace; /* mono：影响代码块的等宽字体。 */
}

```

- 在`Demo\docs-base\.vitepress\theme\style\index.scss`文件中，添加以下内容：

```scss{1}
@use "./font.scss" as *; // 字体
```

- 最后在`Demo\docs-base\.vitepress\theme\index.ts`文件中引入`style\index.scss`文件

```ts{1}
import "./style/index.scss"; // 引入Demo\docs-base\.vitepress\theme\style\index.scss全局样式

```

## 验证

重新启动开发服务器，在浏览器中打开`http://localhost:5173/`，可以看到页面中的字体已经替换为自定义的中文字体。

```sh
pnpm demo:base dev
```
