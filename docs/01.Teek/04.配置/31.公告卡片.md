## 创建组件

- 在`docs\.vitepress\theme\components\`新建`NoticeCard`文件，分别创建以下组件和代码

::: warning
- 由于该组件调用的第三方接口。需自行去[注册账号](https://api.nsmao.net/user/login)，替换实际自己的`API_KEY`

- 代码请在源码自行获取：https://gitee.com/SeasirHyde/teek-hyde/blob/main/docs/.vitepress/theme/components/NoticeCard.vue
:::

```md{5}
├─ docs
│ └─ .vitepress
│ │ └─ theme
│ │ │ └─ components
│ │ │ │ └─ NoticeCard.vue
└─ package.json
```

## 注册组件

- 在`docs\.vitepress\theme\components\TeekLayoutProvider.vue`中注册组件

```vue{2,9}
<script setup lang="ts" name="TeekLayoutProvider">
import NoticeCard from './NoticeCard.vue'  //公告卡片
</script>

<template>
  <Teek.Layout>
        <!-- 自定义卡片 -->
    <template #teek-home-card-my-after>
      <NoticeCard />
    </template>
  </Teek.Layout>
</template>
```
